## Development

Install dependencies:

```sh
npm install
```

Run the project in development mode:

```bash
npm run dev

npm run dev -- --open
```

Running tests:

```sh
npm run test:unit
```

Running linters:

```sh
npm run lint
npm run format
```

## Building

To generate a production version:

```bash
npm run build
./init.sh
```
